# SER225 RPG Game

## What is this?
This is a game created for Quinnipiac's SER225 Agile Development class.
Our team worked on an RPG style game throughout the semester, adding new art, features, and mechanics.
The main character, Gnomeo,  lived a comfortable life in his cabin in the woods, until his love, Juliet, was taken from him. Now, he must fight through the five stages of grief to help rescue her from the clutches of death.

## How to run this project?
This project was bult using Gradle. To run the project, first install Gradle. You can then start the "run" task in Gradle to run the project.

The main method for the game is in the `Game` package > `Game.java` file.<br>
The main method for the map editor is in the `MapEditor` package > `MapEditor.java` file.

## Documentation
